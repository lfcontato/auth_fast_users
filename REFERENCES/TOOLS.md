# Conceitos
- Tokens de Acesso Pessoais (PAT) com escopos e expiração.
- Auditoria por token e limites de uso.

# Endpoints (rascunho)
- POST `/tools/pat` – criar PAT.
- GET `/tools/pat` – listar PATs do usuário.
- DELETE `/tools/pat/{id}` – revogar PAT.
